5. 자바스크립트 & 제이쿼리 제어문 중 조건문 if문의 종류 3가지를 서술하시오 (총15점)

if문
  • 형태: if (조건식) {실행문}
  • 설명: 조건식이 만족하면 그에 해당하는 실행문이 실행된다.
if else문
  • 형태: if (조건식1){실행문1} else {실행문2}
  • 설명: 기본형태의 if 문과 같지만 조건식에 해당하지 않을 경우의 실행문까지 작성가능하다.
if else if문
  • 형태: if (조건식1){실행문1} else if (조건식2){실행문2}
  • 설명: if else문에서 조건식1에 해당하지 않을 경우 새로운 조건식2까지 고려하여 작성가능하다.
